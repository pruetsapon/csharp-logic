Challenge Statement

Logic test is a backend logic application written.

Prerequisites:
- .NET Core SDK 2.1.4

=====================================
Development Environment
=====================================

Logic test application:
- On any terminal move to the "Logic" folder (the folder containing the "Logic.csproj" file) and execute these commands:

dotnet restore
dotnet run

=====================================
To run unit tests
=====================================
- On any terminal move to the "Logic.Test" folder (the folder containing the "Logic.Test.csproj" file) and execute these commands:

dotnet restore
dotnet build
dotnet test
